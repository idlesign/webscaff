# webscaff installation

Use `uv` to install and upgrade `webscaff`.
https://docs.astral.sh/uv/getting-started/installation/

## From sources

```bash
uv tool install --force -e .
```

## From PyPI

### Install

```bash
uv tool install webscaff
```

### Upgrade

```bash
uv tool upgrade webscaff
```
